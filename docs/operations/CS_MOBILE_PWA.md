# CS 운영자 모바일 PWA (Phase 138)

## 접속

`https://kohganepercentiii.com/seller/cs/mobile`

## 홈화면 추가

### iOS
1. Safari에서 위 URL 접속
2. 공유 버튼(↑) → "홈 화면에 추가"
3. 이름 확인 후 "추가"

### Android
1. Chrome에서 위 URL 접속
2. "앱 설치" 배너 또는 메뉴 → "홈 화면에 추가"
3. 또는 페이지 상단 "홈화면 추가" 버튼 클릭

## 기능

- 미응답 메시지 카드 형식 (카테고리별 색상)
- AI 제안 답변 (길게 누르면 textarea에 자동 입력)
- 발송 / 보류 / 해결 버튼
- Service Worker: 정적 자원 오프라인 캐시
